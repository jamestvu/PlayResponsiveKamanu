Playresponsivekamanu

A project using the composite responsive kamanu website as a basis.

Download this project as a .zip file Download this project as a tar.gz file

Overview

This is a sample Play application based on the Responsive Kamanu Website as a basis.

Installation

Download the zip file, unzip the file, cd to the diretory of the file, run the play command, run the eclipse command, and then the "run" command, then you can view the website at http://localhost:9000

This application has none currently.
